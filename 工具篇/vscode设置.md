# vscode 设置

## 快捷键

| 命令                             | 功能 |
| -------------------------------- | ---- |
| workbench.action.navigateForward | 前进 |
| workbench.action.navigateBack    | 后退 |

## 设置

| 命令            | 功能     |
| --------------- | -------- |
| editor.wordWrap | 换行显示 |
